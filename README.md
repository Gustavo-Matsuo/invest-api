# 🚀 InvestTrack API

Uma API RESTful desenvolvida em **Java** com **Spring Boot** para o gerenciamento de ativos financeiros (Ações e Criptomoedas).

## ✨ Funcionalidades:
- **Listar Ativos:** Consulta todos os investimentos cadastrados.
- **Cadastrar Ativos:** Inclusão de novos ativos via requisições JSON.
- **Remover Ativos:** Exclusão de itens específicos por ID.

## 🛠️ Tecnologias Utilizadas:
- **Java 17+**
- **Spring Boot 3**
- **Spring Data JPA**
- **H2 Database** (Banco de dados em memória)

## 🚦 Como executar o projeto:
1. Clone este repositório: `git clone https://github.com/seu-usuario/invest-api.git`
2. Abra no VS Code.
3. Certifique-se de que a porta **8081** está livre.
4. Execute o arquivo `InvestApiApplication.java`.

## 📡 Endpoints Principais:
 Retorna a lista de ativos.
 Cadastra um novo ativo.
 Remove um ativo pelo ID.
